# MDN Theme

目标

Add a theme to docs of MDN.

**给 MDN 的文档换个皮肤。**

`https://developer.mozilla.org/*/docs/*`

工具依赖

- Chrome
- Tampermonkey

技术依赖

- UserScript
- CSS
